PostgreSQL password was auto-generated (provisioning and every consumer share
it). Retrieve it with:
  kubectl get secret {{ include "mcp-mesh-core.postgresCredentialsSecretName" . }} -n {{ .Release.Namespace }} \
    -o jsonpath='{.data.password}' | base64 -d
The secret is reused on upgrade and kept on uninstall so it stays in sync
with the database files on the persistent volume.
