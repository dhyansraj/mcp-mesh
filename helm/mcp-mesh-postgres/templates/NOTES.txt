{{- if include "mcp-mesh-postgres.existingSecret" . }}
PostgreSQL is provisioned with the password from secret
"{{ include "mcp-mesh-postgres.existingSecret" . }}" (key: {{ include "mcp-mesh-postgres.existingSecretPasswordKey" . }}).
{{- else if not (include "mcp-mesh-postgres.password" .) }}
PostgreSQL is provisioned with an auto-generated password. Retrieve it with:

  kubectl get secret {{ include "mcp-mesh-postgres.credentialsSecretName" . }} -n {{ .Release.Namespace }} \
    -o jsonpath='{.data.password}' | base64 -d

The secret is reused on upgrade and kept on uninstall (helm.sh/resource-policy:
keep) so it stays in sync with the database files on the persistent volume.
{{- end }}
