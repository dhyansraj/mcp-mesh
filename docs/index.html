üìö Quick Start

# Install MCP Mesh pip install "mcp-mesh>=0.5,<0.6" # Create your first agent from fastmcp import FastMCP import mesh app = FastMCP("My Service") @app.tool() @mesh.tool(capability="greeting", dependencies=["date_service"]) def greet(date_service=None): return f"Hello! {date_service()}" @mesh.agent(name="my-service", auto_run=True) class MyAgent: pass

That's it! No manual server setup, no connection management, no networking code.
